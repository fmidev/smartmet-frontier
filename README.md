# smartmet-frontier

Part of [SmartMet Server](https://github.com/fmidev/smartmet-server). See the [SmartMet Server documentation](https://github.com/fmidev/smartmet-server) for an overview of the ecosystem.

## Overview

Frontier is an SVG weather chart renderer that reads Weather Object Model (WOML) files and produces styled SVG weather map charts. It is used by FMI to render analysis and forecast charts.

## Features

- Reads WOML (Weather Object Model) files produced by the SmartMet Mirwa editor
- Renders SVG weather charts with meteorological symbols and fronts
- Supports various weather analysis and forecast chart types

## Dependencies

- [smartmet-library-woml](https://github.com/fmidev/smartmet-library-woml) — WOML file reading
